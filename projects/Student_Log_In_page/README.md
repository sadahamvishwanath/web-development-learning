# 🔐 Login & Signup Web App

## 🎯 Objective
To build a **Login and Signup Page** as a hands-on project to **improve my web development skills** and gain practical experience with backend development.

---

## 🚀 About the Project
This project is a simple yet functional **user authentication system** that allows users to:
- Create a new account (Signup)
- Log in with their credentials (Login)
- Log out securely
- Access protected content after authentication

The goal is to understand how real-world authentication systems work, including form handling, database interaction, password security, and session management.

---

## 🛠️ Tech Stack

| Component      | Technology |
|----------------|------------|
| Backend        | Python |
| Database       | SQLite |
| Frontend       | HTML, CSS |
| Authentication | Password hashing & sessions |

---

## ✨ Features

- ✅ User Registration (Signup)  
- ✅ User Login with validation  
- ✅ Secure password hashing (no plain-text storage)  
- ✅ Session management  
- ✅ Logout functionality  
- ✅ Protected routes (only accessible after login)  
- ✅ Flash messages for user feedback

## 📝 Usage

### 🔹 Sign Up
1. Go to the **Signup** page  
2. Enter a username and password  
3. Click **Register**

### 🔹 Log In
1. Go to the **Login** page  
2. Enter your credentials  
3. Access the protected homepage/dashboard  

### 🔹 Log Out
Click the **Logout** button to safely end your session.
