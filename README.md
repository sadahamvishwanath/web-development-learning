## 📌 Notes to Myself

- **Never copy-paste** – type every example manually to build muscle memory.
- **Build, don't just read** – after every 2–3 topics, add a feature to my current project.
- **Commit daily** – even for tiny changes (e.g., "fixed nav alignment").
- **Test responsiveness** – always check the site on both desktop and mobile viewports.
- **Comment your code** – write comments in HTML/CSS to explain *why*, not *what*.
